========================================
  액추에이터 제어 코드 전달 사항
========================================

[ 파일 구조 ]
📁 auto_cycle/
    📄 auto_cycle.ino
📁 push_once/
    📄 push_once.ino


[ 환경 세팅 ]
- 아두이노 IDE 또는 VS Code + Arduino 확장 / PlatformIO
- 보드: Arduino Uno
- 시리얼 모니터 baud: 9600
- 업로드할 때 시리얼 모니터 반드시 닫기


[ 핀 연결 (IBT-2 ↔ 아두이노) ]
IBT-2 핀   |  아두이노 핀
-----------|-----------
RPWM       |  D5
LPWM       |  D6
R_EN       |  D7
L_EN       |  D8
VCC        |  5V
GND        |  GND


[ 사용법 ]

▶ auto_cycle (자동 반복)
  - 's' → 시작 (15초 간격으로 자동 반복)
  - 'q' → 현재 사이클 완료 + 완전 복귀 후 정지

▶ push_once (수동 1회)
  - 'p' → 누를 때마다 전진 → 복귀 1회 실행


[ 주의사항 ]
- USB 업로드 후, 12V 어댑터를 콘센트에 꽂아야 액추에이터 동작
- 12V 어댑터선은 IBT-2 녹색 터미널(B+/B-)에만 연결
  브레드보드/아두이노에 절대 연결 금지
- USB 꽂자마자 자동 동작 안 함
  시리얼 모니터에서 명령어 입력해야 시작
